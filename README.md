# Video Streaming Platform

This project was developed as a final school assignment.

## Description

A full-stack video streaming platform that stores YouTube video URLs in a MongoDB database and displays them dynamically through a REST API. The front-end is built with React to provide a smooth and responsive user interface.

## Tech Stack

-   React
-   JavaScript
-   Node.js
-   Express
-   MongoDB
-   YouTube API

## Features

-   Video catalog management
-   YouTube video embedding
-   REST API integration
-   Responsive UI
